## 지배적 테마

2026-07-04 감사가 `users`에만 적용한 "`organizationId` 불변 강제"가 **차량·운행일지·예약·주유·하이패스·정비의 admin update 경로엔 빠져 있었다**(#6). 더해, 개인 캘린더용 Google OAuth 토큰이 같은 기관 멤버 누구나 읽는 `users` 문서에 저장돼 있었고(#4), 비활성화가 Firestore `status`만 바꿔 Auth·토큰은 살아 있었으며(#1), OCR 트리거가 URL로 지정된 임의 Storage 경로를 Admin SDK로 다운로드했다(#3).
